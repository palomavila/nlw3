## 👩🏽‍💻Projeto

O Happy é um projeto que facilita a visualização de lares que cuidam de crianças que precisam de carinho e atenção.

Desenvolvimento na Next Level Week 3 durante os dias 12 a 18 de Outubro de 2020.

Thanks @RocketSeat 🚀
